# Preuves et captures

Ce dossier est prévu pour recueillir les éléments de preuve utiles à la validation du projet Locatic :

- captures de terminal montrant la construction de l’image Docker
- sorties de commandes kubectl pour vérifier les pods et services
- captures d’écran de l’interface accessible via Nginx
- captures de Grafana ou de Prometheus montrant la supervision du déploiement
- extraits de logs utiles en cas de problème

L’objectif est de garder une trace concrète des étapes de déploiement, de validation et de monitoring réalisées sur la machine locale.
